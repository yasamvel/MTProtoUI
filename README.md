# MTProtoUI

Working MVP MTProto management panel.

## Features

- One-command installation
- RU / EN support
- Classic / DD / EE generator
- tg:// links
- Docker deployment

## Install

```bash
bash <(curl -Ls https://raw.githubusercontent.com/yasamvel/MTProtoUI/main/install.sh)
```

## Remove

```bash
bash <(curl -Ls https://raw.githubusercontent.com/yasamvel/MTProtoUI/main/uninstall.sh)
```
