# MTProtoUI

Working MVP MTProto management panel.

## Features

- One-command install
- RU / EN support
- Dashboard
- Classic / DD / EE generator
- tg:// link generation
- Docker deployment

## Install

bash <(curl -Ls https://raw.githubusercontent.com/yasamvel/MTProtoUI/main/install.sh)

## Manual Start

docker compose up -d --build
